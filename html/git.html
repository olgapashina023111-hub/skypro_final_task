GIT

  • git status или git log - Покажет, какие файлы были изменены, добавлены или удалены в вашем рабочем каталоге и готовы к коммиту.Отобразит историю коммитов вашего проекта, показывая кто, когда и какие изменения внес
  • cd - переход на один уровень вверх.переход в подпапку с именем my_project
  • pwd - текущая рабочая директория".выводит полный путь к текущей папке, в которой находится пользователь.
  • mkdir - используется для создания новых директорий (папок)
  • mv - (переместить) и используется для перемещения или переименования файлов и папок в репозитории.
